Actions Vocabulary v4.0.0

A minimal extension to the Common Core Ontologies (CCO) for tracking action commitments.

Philosophy: Reference CCO classes directly rather than wrapping them. Add only what CCO lacks - lifecycle phase tracking.

Core Concepts

v4 reuses CCO classes directly:

  • Plan (CCO ont00000974): Action definitions / task templates
  • Planned Act (CCO ont00000228): Action instances / executions
  • Objective (CCO ont00000476): Projects / desired outcomes

Custom Extension

Only one custom class added to CCO:

  • ActPhase: Lifecycle states (NotStarted, InProgress, Completed, Blocked, Cancelled)

Properties

  • hasPhase: Links a Planned Act to its current lifecycle phase (functional)
  • hasObjective: Links a Plan to the Objective it serves
  • has_part (BFO): Plan hierarchy
  • prescribes (CCO): Plan ‚Üí Planned Act relationship
