What's Included

  • Core Classes: ActionPlan, ActionProcess, hierarchical structure (Root/Child/Leaf), action states
  • Context Extension: GTD-style contexts (Location, Tool, Energy, Social)
  • Workflow Extension: Dependencies, milestones, blocking relationships, parallel execution
  • Role Integration: Agent assignment, delegation, Areas of Focus (via CCO infrastructure)
  • BFO/CCO Alignment: Formal upper ontology foundation for interoperability
